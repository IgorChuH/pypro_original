# Проект "Виджеты для банка"

## Описание:

Данный проект - это тренировочная работа, направленная на развитие базовых производственных навыков, связанных с профессией Python-разработчик. Тема работы разработка виджета для банковских операций клиента.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/IgorChuH/pypro_original.git
```
2. Установите зависимости:
```
-pip install -r requirements.txt

## Инструкция:

###Главная функция:
1.Для начала вам не обходимо выбрать формат файла (json, csv, xlsx).
2.Введите статус, по которому необходимо выполнить фильтрацию. (EXECUTED, CANCELED, PENDING).
3.Вам предлагаются различные методы сортировки на ваше усмотрение.
4.В конечном итоге выводится список транзакции в удобном формате.


###Отдельные функции
1.Введите номер карты и номер аккаунта для использования маскировки в модуле mask.py
2.Введите данные аккаунта для маскировки в модуле widget.
3.Для фильтрации даты введите словарь с данными.
4 Для сортировки введите список словарей.
5.Если вы хотите воспользоваться функцией filter_by_currency (для фильтрации транзакций по валюте) или transaction_descriptions (для вывода описания транзакции), то воспользуйтесь набором транзайций в модуле conftest.py под название transactions. 
6.Чтобы использовать считывание финансовых операций из CSV- и XLSX-файлов, установите все зависимости из requirements.txt. Далее воспользуйтесь новым модулем и базами данных (src/data_count.py, transactions.csv, transactions_excel.xlsx), чтобы считать финансовые транзакции.
